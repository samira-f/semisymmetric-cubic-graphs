# Computational Exploration of Semisymmetric Cubic Graphs

This project explores symmetry properties of cubic graphs using computational methods.

## Key Concepts
- Graph automorphisms
- Vertex-transitivity
- Edge-transitivity
- Semisymmetric graphs

## Motivation
Inspired by my research in algebraic graph theory, this project bridges theoretical mathematics with computational experiments.

## Features
- Detect automorphisms (brute force)
- Check vertex and edge transitivity
- Analyze cubic graphs
- Visualize graph structures

## Examples
- K3,3
- Cube graph
- Petersen graph

## Future Work
- Cayley graph construction
- Hamiltonian cycle detection
- Efficient automorphism computation
